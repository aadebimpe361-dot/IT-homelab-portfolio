## About This Portfolio

This repository documents hands-on IT home lab projects I built to develop real, practical skills in enterprise IT administration, security analysis, and help desk operations. Each project mirrors tools and workflows used daily in corporate environments.

These labs were built independently — no guided course, no pre-configured templates. Everything was set up from scratch using virtual machines, open-source software, and Microsoft Windows Server.

---

## Projects

### 1. [Active Directory Domain Services Home Lab](./01-active-directory-homelab/)
Built a complete Windows Server domain environment with a domain controller, organizational units, user accounts, security groups, and Group Policy Objects (GPOs). Practiced real-world tasks like user provisioning, password resets, account unlocks, and access control.

**Key tools:** Windows Server, Active Directory, Group Policy Management Console, PowerShell

---

### 2. [Windows Sandbox Troubleshooting Lab](./02-windows-sandbox-lab/)
Deployed Windows Sandbox to safely test software installations, analyze suspicious files, and practice malware risk assessment — all in a disposable, isolated environment that protects the host system.

**Key tools:** Windows Sandbox, Process Monitor (Sysinternals), PowerShell, `.wsb` config files

---

### 3. [Help Desk Ticketing System Practice — osTicket](./03-helpdesk-ticketing-osticket/)
Created and resolved 35 mock support tickets in osTicket, simulating real help desk workflows including ticket intake, prioritization, SLA management, and professional customer communication.

**Key tools:** osTicket, XAMPP/WAMP, Active Directory (for account-related tickets)

## Why I Built These

I'm pursuing a career in IT support and desktop administration. Rather than just earning certifications, I wanted to prove I can actually do the work — setting up the infrastructure, troubleshooting real problems, and documenting everything the way a professional would.

Each lab was designed to answer one question: *"What would I actually do if this ticket came in on my first day?"*
